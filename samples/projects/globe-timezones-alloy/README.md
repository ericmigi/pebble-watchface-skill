# Globe Timezones (Alloy / Moddable)

Pseudo-3D rotating globe watchface for Pebble Alloy SDK projects.

## Features

- Rotating dotted globe projection
- 7 hardcoded cities with UTC offsets
- City markers and labels on visible hemisphere
- Local watch time plus city time table
- Targets only `emery` and `gabbro`

## Build

```bash
npm install
pebble build
pebble install --emulator emery
```


Note: screenshot binaries are not committed in this repository due to PR binary-file restrictions.
